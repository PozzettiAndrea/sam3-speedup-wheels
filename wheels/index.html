SAM3 Speedup Wheels

Pre-built CUDA extension wheels for ComfyUI-SAM3 GPU acceleration.

Quick Install

# Check your versions first
python -c "import torch; print(f'CUDA: {torch.version.cuda}, PyTorch: {torch.__version__}')"

# Then pick the matching wheel index below

Example

# For CUDA 12.4 + PyTorch 2.5.1:
pip install torch_generic_nms cc_torch \
  --find-links https://pozzettiandrea.github.io/sam3-speedup-wheels/cu124-torch251/

Packages

  • torch_generic_nms - GPU-accelerated Non-Maximum Suppression
  • cc_torch - GPU-accelerated Connected Components
